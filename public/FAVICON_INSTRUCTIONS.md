# Archivos de favicon requeridos

Para completar la configuración del favicon, necesitas crear los siguientes archivos en la carpeta public/:

1. favicon.ico (16x16, 32x32, 48x48 pixeles)
2. icon-192.png (192x192 pixeles)  
3. icon-512.png (512x512 pixeles)
4. apple-touch-icon.png (180x180 pixeles)

Estos archivos deben ser creados a partir del logo que me mostraste (el círculo con las banderas de México y Estados Unidos y las alas).

## Instrucciones:
1. Guarda el logo como PNG de alta resolución
2. Usa una herramienta como GIMP, Photoshop, o un convertidor online para crear:
   - favicon.ico: tamaños múltiples (16x16, 32x32, 48x48)
   - icon-192.png: 192x192 pixeles
   - icon-512.png: 512x512 pixeles  
   - apple-touch-icon.png: 180x180 pixeles

## Herramientas recomendadas:
- https://favicon.io/ (para generar favicon desde imagen)
- https://realfavicongenerator.net/ (generador completo de favicons)
